# Task 12: contact.html — Form Layout & Mobile UX

## Requirement

Polish contact.html with form styling and responsive layout.

**Target:** Lighthouse ≥91

## Specs

- **Mobile:** 1-column layout (form above info)
- **Desktop:** 2-column layout (info left, form right, 32px gap)
- **Form labels:** Uppercase, 600 weight, 13px, gold color
- **Form fields:** 16px font, 16px padding, 48px min-height
- **Submit button:** 18px bold, 48px min-height, full-width mobile / auto desktop
- **Contact info card:** Padding 16px, shadow-sm, cream background
- **Form focus:** Gold outline 3px
- **ARIA live regions:** For form errors
- **Accessibility:** Keyboard nav, focus states

## Acceptance Criteria

✅ Mobile 1-column (form over info)
✅ Desktop 2-column (32px gap)
✅ Labels: uppercase, 600, 13px, gold
✅ Fields: 16px, 16px padding, 48px min-height
✅ Button: 18px bold, 48px, full-width mobile
✅ Info card: 16px padding, shadow-sm, cream
✅ Focus: gold 3px outline
✅ ARIA live regions
✅ Keyboard nav
✅ Lighthouse ≥91
✅ Git commit
